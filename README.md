## Overview
A cross platform responsive web application built using React (Gatsby) to convert files from one format to another. The application supports more than 100 different formats 

The application prompts the user to select the file. This auto selects the current file format. The user then chooses the target format and sends the file to the server. 

The linux-based Node.js and Express server then takes the request, converts the format and sends the final formatted file to the user.
